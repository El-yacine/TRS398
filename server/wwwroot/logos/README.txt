LOGO DIRECTORY
==============

Place your clinic/hospital logo image files in this directory.

Supported formats: PNG, JPG, JPEG, GIF, WEBP, BMP

Logo naming:
- For clinic-specific logos: Use your clinic name with spaces replaced by underscores
  Example: "City_Hospital.png" for clinic name "City Hospital"
  
- For default logo: Name the file "logo.png" or "logo.jpg"
  This logo will be used for all clinics that don't have a specific logo

The system will automatically:
1. Look for a logo matching your clinic name
2. Fall back to "logo.png" or "logo.jpg" if no clinic-specific logo is found
3. Use any image file in this directory if no default logo exists

Recommended logo size: 200x200 pixels or larger (will be automatically resized)
